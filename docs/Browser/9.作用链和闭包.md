# 前言
理解作⽤域链是理解闭包的基础，⽽闭包在JavaScript中⼏乎⽆处不在，同时作⽤域和作⽤域链还是所有编程语⾔的基础。所以，作⽤域和作⽤域链⼀定是绕不开的。
这篇文章主要帮助我们了解什么是作⽤域链，并通过作⽤域链再来讲讲什么是闭包。

## 执⾏流程

```js 
function bar() {
    console.log(myName)
}

function foo() {
    var myName = "1111"
    bar()
}

var myName = "2222"

foo()
```
